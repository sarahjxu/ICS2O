This is a calculator application! It can calculate the below functions. Calculations can be done in a string (i.e. 1+2*3/5) or individually (i.e. 1+2=3, 3*2=6). This program does not follow BEDMAS and will calculate numbers and operations in the order they were pressed. Another thing to keep in mind is that trigonometric functions only calculate in degrees, not radians. To use the (-) sign, press it before the first number entered. Happy calculating!
